# 划线 translator

一个简单的浏览器划线翻译插件，将英文划线翻译为中文。

## 功能特点

- 🎯 **划选即翻译**：鼠标选中英文文字，自动显示翻译按钮
- 🌐 **悬浮气泡**：翻译结果以悬浮气泡形式展示，不影响阅读
- 🔑 **百度翻译API**：使用百度翻译的高质量翻译服务
- ⚙️ **易配置**：简洁的设置界面，快速配置API Key

## 安装使用

### 1. 准备百度翻译API

访问 [百度翻译开放平台](http://fanyi.baidu.com) 申请API：

1. 注册百度账号
2. 进入【翻译API】-【立即使用】
3. 创建应用，获取 `APP ID` 和 `密钥`

### 2. 加载插件

1. 克隆或下载本项目到本地
2. 打开浏览器，进入扩展插件页面
3. 开启右上角【开发者模式】
4. 点击【加载已解压的扩展程序】
5. 选择 `translate-plugin` 文件夹

### 3. 配置API

点击插件图标，进入设置页面，输入您申请的：
- APP ID
- 密钥

点击【保存配置】

## 使用方法

1. 在任意网页选中英文文本
2. 文字上方会显示蓝色"翻译"按钮
3. 点击按钮或直接划选后等待
4. 翻译结果将以悬浮气泡形式显示
